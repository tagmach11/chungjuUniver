# 청주대학교 EDELWEIS 교수 대시보드

청주대학교 교수 포털 시스템의 대시보드 페이지를 재현한 프로젝트입니다.

## 📁 파일 구조

```
청주대/
├── index.html      # 메인 HTML 파일
├── style.css       # 스타일시트
├── script.js       # JavaScript 기능
└── README.md       # 프로젝트 설명서
```

## 🎨 주요 기능

### 1. 교수 프로필 카드
- 소속 학과 표시
- 프로필 사진
- 이름 및 직위

### 2. 상담·지도 관리현황
- 학년별 탭 전환 (전체/1학년/2학년/3학년/4학년)
- 상담 미실시자 현황
- 상담 완료 현황
- 방문예약 상담 현황
- 온라인 답변 요청 현황

### 3. 공지사항
- 최신 공지사항 목록
- NEW 배지 표시
- 날짜 표시

### 4. 달력 및 일정
- 월간 달력 뷰
- 이전/다음 달 이동
- 일정이 있는 날짜 표시
- 오늘 날짜 강조
- 하단 일정 목록

### 5. 교수역량진단 & 강의스타일
- 교수역량진단 안내 카드
- 강의스타일 진단 결과 표시

### 6. 교수님 알림
- 중요 공지사항 목록
- 첨부파일 아이콘 표시
- 날짜 표시

## 🚀 실행 방법

### 방법 1: 직접 열기
1. `index.html` 파일을 더블클릭하여 브라우저에서 엽니다.

### 방법 2: 로컬 서버 사용 (권장)
```bash
# Python 3가 설치되어 있는 경우
cd /Volumes/4csoft/청주대
python3 -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

```bash
# Node.js가 설치되어 있는 경우
npx http-server -p 8000

# 브라우저에서 http://localhost:8000 접속
```

## 📝 주요 섹션 설명

### 1. 헤더 (Header)
- 상단: 대학 로고, 사용자 정보, 네비게이션
- 하단: EDELWEIS 로고, 메인 메뉴

### 2. 대시보드 그리드 (Dashboard Grid)
- 3열 레이아웃 (프로필 | 상담현황 | 공지사항)
- 반응형으로 모바일에서는 1열로 전환

### 3. 달력 섹션 (Calendar Section)
- 월간 달력 뷰
- 일정 목록
- 월 이동 버튼

### 4. 하단 정보 카드 (Bottom Sections)
- 2열 그리드 레이아웃
- 교수역량진단 (빨강 카드)
- 강의스타일 (파랑 카드)
- 교수님 알림 (전체 너비)
